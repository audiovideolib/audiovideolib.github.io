Remarks

Strongly-typed properties cover the well-known iTunes atoms; everything else (or anything supplied via the ---- escape hatch) is exposed via Items and FreeFormItems. Serialisation produces the body of an ilst atom (children only — the caller wraps it in ilst / meta / udta / moov).

Properties

Album

Gets or sets the album (©alb).

AlbumArtist

Gets or sets the album artist (aART).

Artist

Gets or sets the artist (©ART).

Bpm

Gets or sets the BPM (tmpo).

Comment

Gets or sets the comment (©cmt).

Compilation

Gets or sets the compilation flag (cpil).

Composer

Gets or sets the composer (©wrt).

CoverArt

Gets the cover art entries (covr); multiple entries are supported.

DiscNumber

Gets or sets the disc number (disk).

DiscTotal

Gets or sets the total number of discs (disk).

FreeFormItems

Gets the dictionary of free-form (----) items keyed by mean + name.

Genre

Gets or sets the genre. Sourced from ©gen if present, otherwise a numeric mapping from gnre (1-based ID3v1 genre index, surfaced as the raw integer in string form).

Items

Gets all parsed ilst items, in the order they appeared in the source atom.

Title

Gets or sets the title (©nam).

Tool

Gets or sets the encoding tool / encoder (©too).

TrackNumber

Gets or sets the track number (trkn).

TrackTotal

Gets or sets the total number of tracks (trkn).

Year

Gets or sets the year / release date (©day).

Methods

Parses the body of an ilst atom (children only — no surrounding ilst header) into a tag.

SetFreeFormItem(string, string, string?)

Sets or removes a free-form item identified by its mean namespace and name key.

ToByteArray()

Serialises the tag's strongly-typed properties and free-form items into the body of an ilst atom (children only — the caller is responsible for wrapping the result in an ilst + meta + udta + moov chain).

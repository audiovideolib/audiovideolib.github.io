WriteString(string?, Encoding, int)

Writes a truncated string to the buffer using the encoding provided.

public void WriteString(string? value, Encoding encoding, int maxBytesToWrite)

Parameters

value string

The string.

encoding Encoding

The encoding to encode the string as.

maxBytesToWrite int

The max amount of bytes allowed to write to the buffer.

Remarks

If the amount of bytes of the encoded string exceeds the maxBytesToWrite parameter, the function will remove the last character of the string, and encode it again. It will do this as long as the amount of bytes of the encoded string exceeds the maxBytesToWrite parameter. This is useful if you have a long string but are limited to the amount of bytes you can use.

This function does NOT write a string terminator character at the end if it's not appended to the string!

Exceptions

NotSupportedException

The stream does not support writing, or the stream is already closed.

ObjectDisposedException

Methods were called after the stream was closed.

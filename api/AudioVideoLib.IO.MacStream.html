Structural walker for Monkey's Audio (.ape) files (version 3.98+ — files with the APE_DESCRIPTOR + APE_HEADER layout). Surfaces the descriptor, header, seek table, and per-frame byte ranges. WriteTo(Stream) is byte-passthrough — no audio re-encoding happens at any point. APEv2 footer / ID3v1 footer tag editing flows through the existing AudioTags scanner; this walker only handles the audio container.

Remarks

Both integer ("MAC ") and float ("MACF") variants are accepted; the active variant is surfaced via Format. See 3rdparty/MAC_1284_SDK/Source/MACLib/APECompressCreate.cpp:250-253 for how the upstream encoder writes the cID. Pre-3.98 files (APE_HEADER_OLD, no descriptor) are not supported — ReadStream(Stream) returns false for them.

ReadStream(Stream) populates _source with a StreamSourceReader that holds the supplied System.IO.Stream open via leaveOpen: true; callers must keep that System.IO.Stream alive until WriteTo(Stream) finishes, in line with the source-stream lifetime contract on IMediaContainer.

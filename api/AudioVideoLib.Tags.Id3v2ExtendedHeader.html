Class Id3v2ExtendedHeader

Namespace
AudioVideoLib.Tags
Assembly
AudioVideoLib.dll

The extended header contains information that can provide further insight in the structure of the Id3v2Tag, but is not vital to the correct parsing of the Id3v2Tag information; hence the extended header is optional.

Remarks

The walker captures the fmt and data chunks needed to report format parameters and locate the sample block, and records every other chunk structurally without materialising its payload. Only the WAVE form type is recognised; other RIFF types such as AVI are ignored.

Gets the four-character RIFF form type. Only "WAVE" is currently recognised.

InfoTag

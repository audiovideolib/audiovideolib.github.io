ReadString(int)

Read amount of bytes from the stream as a string using the System.Text.Encoding.ASCII encoding.

public string ReadString(int lengthBytes)

Parameters

lengthBytes int

The amount of bytes to read.

Returns

string

A string from the stream decoded using the System.Text.Encoding.ASCII encoding. Returns an empty string if the end of file has been reached. If there's a byte order marker at the start of the stream, it will be used to indicate the encoding to use.

Exceptions

NotSupportedException

The stream does not support reading.

ObjectDisposedException

Methods were called after the stream was closed.

ReadString(Encoding)

Read a string from the stream in the specified encoding until the string NULL-terminator is read or the end of file is reached.

public string ReadString(Encoding encoding)

Parameters

encoding Encoding

The encoding used to read the bytes as.

Returns

string

A string from the stream decoded using the encoding encoding.

Remarks

Returns an empty string if the end of file has been reached. The string NULL-terminator is also consumed but not appended to the string. The string NULL-terminator is the '\0' character encoded with the encoding. If there's a byte order marker at the current position of the stream it will be used to indicate the encoding to use (overriding the given encoding).

Exceptions

NotSupportedException

The stream does not support reading.

ObjectDisposedException

Methods were called after the stream was closed.

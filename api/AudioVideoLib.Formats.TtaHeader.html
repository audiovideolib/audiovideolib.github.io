Remarks

Mirrors the TTA_info struct in 3rdparty/libtta-c-2.3/libtta.h, plus the 4-byte trailing CRC32 that read_tta_header in libtta.c consumes immediately after the five info fields. All multi-byte integers are little-endian.

Fields

FixedSize

Size of the fixed header on disk, in bytes (4-byte magic + 18 bytes of info + 4-byte CRC).

Properties

BitsPerSample

Bits per sample (libtta range: MIN_BPS = 16 through MAX_BPS = 24).

Format

Audio format code: 1 = simple, 2 = encrypted (libtta TTA_FORMAT_*).

HeaderCrc32

CRC32 stored at offsets [18..22) of the fixed header.

Magic

The four-byte magic at offset 0: T T A 1.

NumChannels

Channel count (libtta caps at MAX_NCH = 6).

SampleRate

Sample rate in Hz (libtta sps).

TotalSamples

Total decoded sample count across the whole stream.

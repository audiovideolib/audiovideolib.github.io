Read(long, Span<byte>)

Reads bytes starting at offset into destination. Reads exactly destination.Length bytes; throws System.IO.EndOfStreamException if fewer are available.

public void Read(long offset, Span<byte> destination)

Parameters

offset long

The absolute byte offset (from the start of the source).

destination Span<byte>

The buffer to read into.

Exceptions

ArgumentOutOfRangeException

If offset is negative or beyond Length.

EndOfStreamException

If the source has fewer than destination.Length bytes available from offset.

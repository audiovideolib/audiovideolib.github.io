Remarks

The walker captures the DSD , fmt , and data chunks (in that order, per spec) and locates an optional ID3v2 metadata block at the absolute offset given by the DSD chunk's metadata-pointer field. All multi-byte integers in DSF are little-endian.

Properties

BitsPerSample

Gets the bits-per-sample value from the fmt chunk (1 or 8).

BlockSizePerChannel

Gets the per-channel block size from the fmt chunk (typically 4096).

ChannelType

Gets the channel-type code from the fmt chunk (1=mono, 2=stereo, 3=3ch, 4=quad, 5=4ch, 6=5ch, 7=5.1ch).

Channels

Gets the channel count from the fmt chunk.

Chunks

Gets the chunks discovered inside the DSF container, in file order.

DataOffset

Gets the absolute offset of the data chunk's payload, or 0 if the chunk is absent.

DataSize

Gets the size of the data chunk's payload in bytes (excludes the 12-byte chunk header).

DeclaredFileSize

Gets the declared total file size from the DSD chunk, in bytes.

EmbeddedId3v2

Gets the embedded ID3v2 tag, or null when no metadata block is present or it could not be parsed.

EndOffset

Gets the end offset of the IMediaContainer, where it ends in the stream.

FormatId

Gets the format identifier from the fmt chunk (0 = DSD raw).

FormatVersion

Gets the format version from the fmt chunk (typically 1).

MaxFrameSpacingLength

Gets or sets the max length of spacing, in bytes, between 2 frames when searching for frames.

MetadataPointer

Gets the absolute offset of the embedded ID3v2 metadata block, or 0 if no metadata is declared.

SampleCount

Gets the per-channel sample count from the fmt chunk.

SampleRate

Gets the sample rate from the fmt chunk, in Hz (e.g. 2822400 for DSD64).

StartOffset

Gets the start offset of the IMediaContainer, where it starts in the stream.

TotalDuration

Gets the total length of audio in milliseconds.

TotalMediaSize

Gets the total size of audio data in bytes.

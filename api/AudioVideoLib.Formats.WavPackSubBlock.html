Remarks

Sub-block IDs are documented in 3rdparty/WavPack/include/wavpack.h:158-193. ID_LARGE (0x80) and ID_ODD_SIZE (0x40) are flag bits; the unique ID occupies the lower 6 bits (mask 0x3F). ID_OPTIONAL_DATA (0x20) is part of that unique-ID space — for example ID_RIFF_HEADER = 0x21 = ID_OPTIONAL_DATA | 1. Callers can compare RawId (or UniqueId) directly against the named constants.

Properties

IsLargeSize

Gets a value indicating whether the ID_LARGE bit is set (i.e., the on-disk size header was 24-bit).

IsOptional

Gets a value indicating whether the ID_OPTIONAL_DATA band is set.

PayloadLength

Gets the logical payload length in bytes — already accounts for ID_ODD_SIZE.

PayloadOffset

Gets the file offset (within the source stream) of the first payload byte.

RawId

Gets the raw id byte (including the ID_LARGE and ID_ODD_SIZE flag bits).

UniqueId

Gets the unique id with the two flag bits cleared (bit 7, bit 6). The ID_OPTIONAL_DATA bit (0x20) is part of the unique-id space and is preserved.

Methods

ReadPayload()

Reads the sub-block payload bytes from the source. The source must be alive — after the owning WavPackStream is disposed, this method throws.

EncodeId(long)

Encodes a raw EBML element id (including its existing marker bit) as its on-disk VINT bytes.

EncodeUInt(long)

Encodes a non-negative unsigned integer payload using the minimum number of bytes (1..8). This is the EBML payload encoding for uint elements (e.g. TargetTypeValue).

EncodeVintSize(long)

Encodes a non-negative integer as an EBML VINT using the smallest legal length (1..8 bytes). The marker bit is included in the output.

EncodeVintSize(long, int)

Encodes a non-negative integer as an EBML VINT of the requested length (1..8 bytes), with marker bit. Useful when forcing a specific encoding length (e.g. for round-trip tests).

TryReadVintId(Stream, out int, out long)

Reads an EBML variable-length integer used for element ids — the marker bit is preserved.

TryReadVintSize(Stream, out int, out long, out bool)

Reads an EBML variable-length integer used for sizes / unsigned ints — the marker bit is stripped.

Remarks

WavPack files are a flat concatenation of blocks; every block begins with this header. The walker reads the header to discover block size (so it can seek to the next block) and to interpret the flags bitfield (sample rate index, channel count, bytes-per-sample). The 40-bit BlockIndex and TotalSamples fields combine the low 32-bit field with the high 8-bit "u8" extension byte, matching the GET_BLOCK_INDEX / GET_TOTAL_SAMPLES macros in wavpack.h:82-95.

Fields

Size

Size of the on-disk header in bytes (always 32).

Properties

BlockIndex

Gets the combined 40-bit block index per GET_BLOCK_INDEX.

BlockIndexHigh

Gets the high 8 bits of the 40-bit block index.

BlockIndexLow

Gets the low 32 bits of the block-index field.

BlockSamples

Gets the number of audio samples encoded in this block.

BytesPerSample

Gets the bytes per sample (1-4) decoded from the BYTES_STORED field — (flags & 3) + 1.

CkId

Gets the four-byte chunk ID; always "wvpk" for valid blocks.

CkSize

Gets the block payload size in bytes — i.e., ckSize. The total on-disk block size is CkSize + 8 because ckSize excludes the leading 8 bytes of the header (per the WavPack file-format documentation referenced in wavpack.h:62-67).

Crc

Gets the block CRC.

Flags

Gets the flags bitfield. See wavpack.h:108-141 for bit definitions.

IsDsd

Gets a value indicating whether the block is encoded DSD (DSD_FLAG, wavpack.h:141).

IsFinalBlock

Gets a value indicating whether this is the final block of a multichannel segment (FINAL_BLOCK).

IsFloat

Gets a value indicating whether the block stores IEEE 32-bit float samples (FLOAT_DATA, wavpack.h:116).

IsHybrid

Gets a value indicating whether the block is from a hybrid-mode encoding (HYBRID_FLAG, wavpack.h:112).

IsInitialBlock

Gets a value indicating whether this is the initial block of a multichannel segment (INITIAL_BLOCK).

IsMono

Gets a value indicating whether the block represents mono audio (MONO_FLAG, wavpack.h:111).

SampleRate

Gets the sample rate in Hz, or 0 if the rate index is the reserved "non-standard" value (15). Decoded from the SRATE_MASK bits at SRATE_LSB == 23 per wavpack.h:131-132; index table from 3rdparty/WavPack/src/common_utils.c:31-32.

TotalSamples

Gets the combined 40-bit total-samples value, or -1 if unknown (low 32 bits == 0xFFFFFFFF) per GET_TOTAL_SAMPLES.

TotalSamplesHigh

Gets the high 8 bits of the 40-bit total-samples field.

TotalSamplesLow

Gets the low 32 bits of total samples; combined with TotalSamplesHigh per GET_TOTAL_SAMPLES (wavpack.h:94). A value of 0xFFFFFFFF in the low 32 bits signals "unknown" — surfaced as TotalSamples == -1.

Version

Gets the stream version. Library handles 0x402..0x410 per wavpack.h:146-147.

Methods

Parse(ReadOnlySpan<byte>)

Parses the 32-byte preamble. Returns null if span is shorter than Size or its first four bytes are not the ASCII magic "wvpk".

MediaType

Gets or sets the type of the media.

public Id3v2TextFrame? MediaType { get; set; }

Property Value

Id3v2TextFrame

The type of the media.

Remarks

The 'Media type' frame describes from which media the sound originated. This may be a text string or a reference to the predefined media types found in the list below. References are made within "(" and ")" and are optionally followed by a text refinement, e.g. "(MC) with four channels". If a text refinement should begin with a "(" character it should be replaced with "((" in the same way as in the "TCO" frame. Predefined refinements is appended after the media type, e.g. "(CD/S)" or "(VID/PAL/VHS)".

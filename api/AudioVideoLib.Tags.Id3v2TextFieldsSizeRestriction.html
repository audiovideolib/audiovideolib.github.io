Id3v240 Id3v2Tag text fields size restrictions.

public enum Id3v2TextFieldsSizeRestriction

Fields

Max1024Characters = 1

No string is longer than 1024 characters.

Max128Characters = 2

No string is longer than 128 characters.

Max30Characters = 3

No string is longer than 30 characters.

NoRestrictions = 0

No restrictions.

Remarks

Note that nothing is said about how many bytes is used to represent those characters, since it is encoding dependent. If a text frame consists of more than one string, the sum of the strings is restricted as stated.

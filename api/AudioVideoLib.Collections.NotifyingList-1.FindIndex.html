FindIndex(Predicate<T>)

Searches for an element that matches the conditions defined by the specified predicate, and returns the zero-based index of the first occurrence within the entire NotifyingList<T>.

public int FindIndex(Predicate<T> match)

Parameters

match Predicate<T>

The Predicate<T> delegate that defines the conditions of the element to search for.

Returns

int

The zero-based index of the first occurrence of an element that matches the conditions defined by match, if found; otherwise, -1.

Exceptions

ArgumentNullException

match is null.

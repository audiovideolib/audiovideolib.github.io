Remarks

Provides serialisation back to the Matroska Tags element bytes for round-trip writers, plus a small set of strongly-typed accessors that look up well-known SimpleTag names by EBML metadata convention.

Fields

AlbumLevel

The Matroska album-level TargetTypeValue.

TrackLevel

The Matroska track-level TargetTypeValue.

Properties

Album

Gets the album name (album-level only).

Artist

Gets the artist (album-level preferred, track-level fallback).

Comment

Gets the comment (album-level preferred, track-level fallback).

Date

Gets the recording date (album-level preferred, track-level fallback).

Entries

Gets the parsed Tag entries, one per top-level Tag element in the segment.

Genre

Gets the genre (album-level preferred, track-level fallback).

Title

Gets the album title, falling back to the track title if no album-level title exists.

TrackNumber

Gets the track number (track-level only by convention).

Methods

FromPayload(byte[])

Parses a Tags element's payload bytes (the bytes after its id+size header) into a MatroskaTag.

SerializePayload()

Serialises just the payload bytes of the Tags element (i.e. the concatenation of every Tag entry).

ToByteArray()

Serialises this tag as the on-disk bytes of a Matroska Tags element (id + size VINT + payload).

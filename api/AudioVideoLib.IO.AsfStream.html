Remarks

The walker validates the top-level Header Object GUID and enumerates its child objects, recording each one as an AsfObject. Recognised metadata-bearing children (Content Description, Extended Content Description, Metadata, Metadata Library — the last two nested inside the Header Extension Object) are parsed into MetadataTag. Unrecognised objects are recorded generically and skipped without failing the walk.

Only the (small) ASF Header Object is materialised in memory; the much larger Data Object and any Index Object that follow are kept reachable via an ISourceReader reference and streamed straight to the destination at write time. Editing metadata in a multi-GB WMV is bounded by the header size, not the file size.

Callers must keep the source System.IO.Stream passed to ReadStream(Stream) alive until WriteTo(Stream) / ToByteArray finishes.

Properties

EndOffset

Gets the end offset of the IMediaContainer, where it ends in the stream.

HeaderObjectChildrenOffset

Gets the absolute offset of the Header Object payload (the byte immediately after its 30-byte header + 6-byte child-count + reserved fields). 0 if the file was not parsed.

HeaderObjectSize

Gets the size of the Header Object as recorded in its on-disk size field, or 0.

MaxFrameSpacingLength

Gets or sets the max length of spacing, in bytes, between 2 frames when searching for frames.

MetadataTag

Gets the parsed ASF metadata tag — the union of the Content Description, Extended Content Description, Metadata, and Metadata Library objects discovered inside the Header Object.

Objects

Gets every ASF object encountered by the walker, in file order. The first entry is always the Header Object itself; the remainder are its direct children. Nested children of the Header Extension Object (Metadata, Metadata Library) are not exposed here.

PlayDuration100Ns

Gets the file's Play Duration as reported by the File Properties Object, in 100-nanosecond units. 0 if the File Properties Object is absent.

StartOffset

Gets the start offset of the IMediaContainer, where it starts in the stream.

TotalDuration

Gets the total length of audio in milliseconds.

TotalMediaSize

Gets the total size of audio data in bytes.

Methods

Dispose()

Releases the underlying ISourceReader. Does not close the user's source System.IO.Stream; the caller still owns that.

ReadStream(Stream)

Reads the audio stream from the stream.

WriteTo(Stream)

Writes the serialised form of this container to the supplied destination.

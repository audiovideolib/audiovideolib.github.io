Remarks

SV7 uses "frames", SV8 uses keyed packets. We unify on "packet". For SV7, Key is null; for SV8 it is the 2-character ASCII key (e.g. "SH", "AP", "RG", "EI", "SE", "ST", "SO", "CT"). See 3rdparty/musepack_src_r475/libmpcdec/mpc_demux.c (the KEY_* macros and mpc_demux_header at line 477) for the full set.

Properties

Key

Gets the SV8 2-character ASCII packet key, or null for SV7 frames.

Length

Gets the total length of this packet in bytes (header plus payload).

SampleCount

Gets the number of decoded audio samples this packet contributes (0 for non-audio packets).

StartOffset

Gets the byte offset of this packet from the start of the source stream.

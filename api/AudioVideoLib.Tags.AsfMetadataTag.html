Properties

Author

Gets or sets the Author field from the Content Description Object.

Copyright

Gets or sets the Copyright field from the Content Description Object.

Description

Gets or sets the Description field from the Content Description Object.

Extended

Gets the Extended Content Description Object items as a dictionary keyed by name. When a name appears more than once the first occurrence wins; use ExtendedItems for the full list.

ExtendedItems

Gets the parsed Extended Content Description Object items as an ordered list of name/value pairs. Duplicate names are preserved in file order.

HasContentDescription

Gets a value indicating whether any field of the Content Description Object is populated.

MetadataItems

ParseContentDescription(byte[])

Parses a Content Description Object payload (after the 24-byte object header) into this tag. Sets Title, Author, Copyright, Description, and Rating. Returns silently when the payload is too short to hold the five length WORDs.

ParseContentDescription(ReadOnlySpan<byte>)

Parses a Content Description Object payload (after the 24-byte object header) into this tag.

ParseExtendedContentDescription(byte[])

Parses an Extended Content Description Object payload (after the 24-byte object header). Items are appended to ExtendedItems in file order. Truncated payloads stop the walk early without throwing.

ToByteArrays()

Serialises the tag's four metadata-bearing objects to byte arrays in canonical order: CDO, ECDO, MO, MLO. Each entry is a complete ASF object (24-byte header + payload). Objects whose source collection is empty are omitted.

U16Le(ushort)

Returns a 2-byte little-endian encoding of value.

U32Le(uint)

Returns a 4-byte little-endian encoding of value.

U64Le(ulong)

Returns an 8-byte little-endian encoding of value.

WrapObject(Guid, byte[])

Wraps a payload in a complete ASF object header (16-byte GUID + 8-byte little-endian size).

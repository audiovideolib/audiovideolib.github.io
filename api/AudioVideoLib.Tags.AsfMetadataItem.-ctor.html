A single Metadata Object (MO) or Metadata Library Object (MLO) item.

public AsfMetadataItem(ushort LanguageListIndex, ushort StreamNumber, string Name, AsfTypedValue Value)

Parameters

LanguageListIndex ushort

Language list index (always 0 for the Metadata Object; may be non-zero for the Metadata Library Object). Stored on disk in the "reserved / language list index" WORD that precedes the stream number.

StreamNumber ushort

The 0-127 stream number this item is associated with, or 0 to mean "any stream".

Name string

The item name (UTF-16LE on disk).

Value AsfTypedValue

The typed value.

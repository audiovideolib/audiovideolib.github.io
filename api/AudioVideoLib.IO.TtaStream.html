Walker for TrueAudio (.tta) streams. Parses the TTA1 fixed header and the per-frame seek table, exposing the frame layout for inspection. WriteTo(Stream) streams the original bytes verbatim from the source — no audio re-encoding.

Remarks

Format reference: 3rdparty/libtta-c-2.3/libtta.c, especially read_tta_header (lines 449-471) and tta_decoder_read_seek_table (lines 473-490). The standard frame length in samples is libtta's MUL_FRAME_TIME(sps) = 256 * sps / 245 (libtta.c:274).

ReadStream(Stream) populates _source with a StreamSourceReader that holds the supplied System.IO.Stream open via leaveOpen: true; callers must keep that System.IO.Stream alive until WriteTo(Stream) finishes, in line with the source-stream lifetime contract on IMediaContainer.

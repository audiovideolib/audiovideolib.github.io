Class Id3v2TagRestrictions

Namespace
AudioVideoLib.Tags
Assembly
AudioVideoLib.dll

For some applications it might be desired to restrict a tag in more ways than imposed by the Id3v2 specification. Note that the presence of these restrictions does not affect how the tag is decoded, merely how it was restricted before encoding.

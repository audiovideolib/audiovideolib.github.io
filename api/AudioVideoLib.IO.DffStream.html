Remarks

The walker captures top-level chunks inside the FRM8 form (form-type "DSD "), descends into the PROP form to extract FS (sample rate) and CHNL (channel count), and locates optional DIIN, COMT, and ID3 chunks. All multi-byte integers in DFF are big-endian.

Properties

BitsPerSample

Gets the bits-per-sample value (DSD is always 1).

ChannelIds

Gets the channel identifiers from PROP/CHNL, one four-CC per channel.

Channels

Gets the channel count from PROP/CHNL.

Chunks

Gets the top-level chunks discovered inside the FRM8 form, in file order.

ComtChunk

Gets the raw payload of the optional COMT comments chunk, or null when absent.

DataOffset

Gets the absolute offset of the DSD audio chunk's payload, or 0 if the chunk is absent.

DataSize

Gets the size of the DSD audio chunk's payload in bytes.

DiinChunk

Gets the raw payload of the optional DIIN info chunk, or null when absent.

EmbeddedId3v2

Gets the embedded ID3v2 tag from the optional ID3 chunk, or null when absent or unparseable.

EndOffset

Gets the end offset of the IMediaContainer, where it ends in the stream.

FormatType

Gets the form-type four-CC (always "DSD " for a recognised DFF file).

FormatVersion

Gets the format version from the FVER chunk (high byte = major).

MaxFrameSpacingLength

Gets or sets the max length of spacing, in bytes, between 2 frames when searching for frames.

SampleCount

Gets the per-channel sample count, computed from the DSD audio chunk size and channel count.

SampleRate

Gets the sample rate from PROP/FS , in Hz.

StartOffset

Gets the start offset of the IMediaContainer, where it starts in the stream.

TotalDuration

Gets the total length of audio in milliseconds.

TotalMediaSize

Gets the total size of audio data in bytes.

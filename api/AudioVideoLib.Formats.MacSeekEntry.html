A single entry from the Monkey's Audio seek table — the absolute file offset of an APE frame's first byte. The seek table is a packed array of 32-bit little-endian offsets; element count is SeekTableBytes / 4. See 3rdparty/MAC_1284_SDK/Source/MACLib/APEHeader.cpp:268.

Remarks

The on-disk offsets are 32-bit, but for files larger than 4 GiB the upstream encoder allows them to wrap. MacStream applies the same wrap-correction as Convert32BitSeekTable (APEHeader.cpp:116-131) — accumulating 0x1_0000_0000 each time the raw value decreases — so FileOffset is widened to long and represents the corrected absolute offset.

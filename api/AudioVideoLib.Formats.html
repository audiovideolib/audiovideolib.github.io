MacDescriptor

The Monkey's Audio file descriptor — the first structure in every APE file (3.98+). Field layout mirrors APE_DESCRIPTOR in 3rdparty/MAC_1284_SDK/Source/MACLib/MACLib.h:179-194.

MacFrame

A single Monkey's Audio frame — a contiguous byte range in the source file plus the number of audio blocks it encodes. Frame boundaries are derived from the seek table; the final frame's length is bounded by TotalApeFrameDataBytes.

MacHeader

The Monkey's Audio per-stream header — fixed-size structure that follows the MacDescriptor. Field layout mirrors APE_HEADER in 3rdparty/MAC_1284_SDK/Source/MACLib/MACLib.h:199-211.

MacSeekEntry

A single entry from the Monkey's Audio seek table — the absolute file offset of an APE frame's first byte. The seek table is a packed array of 32-bit little-endian offsets; element count is SeekTableBytes / 4. See 3rdparty/MAC_1284_SDK/Source/MACLib/APEHeader.cpp:268.

Remarks

Source-reference lifetime model per IMediaContainer: ReadStream(Stream) records per-packet offsets/lengths, WriteTo(Stream) streams those byte ranges directly from source to destination via CopyTo(long, long, Stream). No audio re-encoding; tag handling (APEv2 footer, ID3v2 header) is delegated to AudioTags. The walker is robust to a non-zero StartOffset when the caller has advanced past an ID3v2 prefix.

Properties

EndOffset

Gets the end offset of the IMediaContainer, where it ends in the stream.

Header

Gets the parsed Musepack stream header, or null if ReadStream(Stream) has not succeeded.

MaxFrameSpacingLength

Gets or sets the max length of spacing, in bytes, between 2 frames when searching for frames.

Packets

Gets the list of packets discovered in the container, in file order. SV7 produces a single audio-span packet; SV8 produces one entry per keyed packet.

StartOffset

Gets the start offset of the IMediaContainer, where it starts in the stream.

TotalDuration

Gets the total length of audio in milliseconds.

TotalMediaSize

Gets the total size of audio data in bytes.

Version

Gets the bitstream version detected at ReadStream(Stream) time. Defaults to Sv8 before a successful read.

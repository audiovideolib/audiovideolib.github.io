Remarks

The walker captures the COMM chunk needed to report format parameters and locates the SSND chunk, while recording every other chunk structurally without materialising its payload. Sample rate is decoded from the 80-bit IEEE 754 extended-precision value stored by the AIFF spec.

Properties

Channels

Gets the channel count from the COMM chunk.

Chunks

Gets the chunks discovered inside the AIFF container, in file order.

Compression

Gets the four-character compression identifier from an AIFF-C COMM chunk, or null for plain AIFF.

EndOffset

Gets the end offset of the IMediaContainer, where it ends in the stream.

FormatType

Gets the four-character form type, either "AIFF" or "AIFC".

MaxFrameSpacingLength

Gets or sets the max length of spacing, in bytes, between 2 frames when searching for frames.

SampleFrames

Gets the total number of sample frames from the COMM chunk.

SampleRate

Gets the sample rate in Hz, decoded from the 80-bit IEEE 754 extended-precision value in the COMM chunk.

SampleSize

Gets the number of bits per sample from the COMM chunk.

SsndOffset

Gets the absolute offset of the SSND chunk's payload, or 0 if the chunk is absent.

SsndSize

Gets the size of the SSND chunk's payload in bytes.

StartOffset

Gets the start offset of the IMediaContainer, where it starts in the stream.

TextChunks

Gets the bundle of standard AIFF text chunks (NAME, AUTH, ANNO, COMT) discovered while walking the container, or null if none were present.

TotalDuration

Gets the total length of audio in milliseconds.

TotalMediaSize

Gets the total size of audio data in bytes.

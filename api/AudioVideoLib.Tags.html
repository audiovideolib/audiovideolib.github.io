Id3v1Tag

Per-field text storage for an ID3v1 tag. The source of truth is the raw byte sequence captured at parse time; a per-field Encoding override controls how those bytes are decoded and (on save) how new string values are re-encoded. This lets callers display a tag written with one encoding (e.g. UTF-8) even when the tag-level default is something else (e.g. Latin-1).

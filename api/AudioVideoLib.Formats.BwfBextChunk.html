Remarks

Layout (little-endian): Description[256], Originator[32], OriginatorReference[32], OriginationDate[10], OriginationTime[8], TimeReference (uint64), Version (uint16), UMID[64], (v1+) loudness fields (5 × int16), Reserved[180] (v1) / [180] (v2 same pad), CodingHistory (rest of chunk, ASCII).

Fields

MinV0Size

The minimum size of a v0 bext chunk payload, in bytes.

MinV1Size

The minimum size of a v1+ bext chunk payload (adds 10 bytes of loudness fields, removes 10 from reserved).

Properties

CodingHistory

Gets or sets the coding history (CRLF-separated ASCII, terminates the chunk).

Description

Gets or sets the description (max 256 ASCII chars).

LoudnessRange

Gets or sets the loudness range (×100, v1+).

LoudnessValue

Gets or sets the integrated loudness value (×100, v1+).

MaxMomentaryLoudness

Gets or sets the maximum momentary loudness (×100, v1+).

MaxShortTermLoudness

Gets or sets the maximum short-term loudness (×100, v1+).

MaxTruePeakLevel

Gets or sets the maximum true peak level (×100, v1+).

OriginationDate

Gets or sets the origination date (10 ASCII chars, "YYYY-MM-DD").

OriginationTime

Gets or sets the origination time (8 ASCII chars, "HH:MM:SS").

Originator

Gets or sets the originator (max 32 ASCII chars).

OriginatorReference

Gets or sets the originator reference (max 32 ASCII chars).

TimeReference

Gets or sets the time reference (samples since midnight) as a 64-bit value.

Umid

Gets or sets the SMPTE UMID (always 64 bytes; padded/truncated on write).

Version

Gets or sets the bext version (0, 1, or 2).
